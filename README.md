# Contact List CRUD - DOM + LocalStorage + GitHub

## Estructura por fases
- `fase-1/`: maquetación + crear y listar.
- `fase-2/`: agrega editar y eliminar.
- `fase-3/`: versión final con CRUD completo, spinner, validaciones y LocalStorage.
- `assets/`: recurso SVG principal.

## Requisitos cumplidos
- HTML semántico
- CSS con nombres BEM
- JavaScript Vanilla
- CRUD completo
- LocalStorage
- Validación para evitar datos vacíos
- Spinner / mensaje de carga
- Responsive

## Archivo recomendado para entregar
Usa la carpeta `fase-3/`.
